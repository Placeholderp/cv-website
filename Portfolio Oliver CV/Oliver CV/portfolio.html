Portfolio 1 - Gradient background generator website

In this project I created a website using HTML, CSS and JavaScript. This website allows the users to either choose two rgb values to create a colour gradient or click a button to randomly select two. This has been helpful in allowing me to quickly and easily test how different colour combinations would look like on a website.

In this project, I created a videogame in GameMaker studio. It is a platformer containing all basic game components like lives, score etc... and varying enemies with a boss at the end of each level.
